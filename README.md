## ✨ Premium Features

### 🤖 BloomBot: AI Shopping Assistant
- **Context-Aware Recommendations:** Integrated **OpenAI** GPT-3.5/4 models that understand user queries and inventory.
- **Smart Retrieval (RAG):** Filters products based on natural language search and user budget.
- **Order Tracking AI:** Can interpret order status and explain tracking details to users.

### 🔔 Multi-Channel Notification Mesh
- **Branded HTML Emails:** Automated workflows for registration using SMTP.
- **Real-time SMS Alerts:** Powered by **Twilio** for order confirmation and OTPs.
- **WhatsApp Integration:** Direct customer engagement via Twilio WhatsApp API.
- **In-App Notifications:** Real-time push notifications for order updates.

### 💳 Financial Infrastructure
- **Secure Payments:** Full **Razorpay** integration with signature verification.
- **Automated Invoicing:** Dynamic PDF invoice generation using **ReportLab**.
- **Role-Based Portals:** Dedicated dashboards for **Suppliers (Sellers)** and **Consumers**.

## 🏗 Technology Stack

- **Backend:** Python 3.11, Django 6.x, DRF, PostgreSQL (Production) / SQLite (Dev)
- **Frontend:** React 19, Vite, Axios, React-Router-Dom, Recharts
- **DevOps:** Render (PaaS), GitHub Actions, WhiteNoise (Static serving)
- **APIs:** OpenAI, Twilio, Razorpay

## 🚀 Rapid Deployment (on Render)

This project is configured for **Zero-Config Deployment** using the provided `render.yaml`.

1. **Connect Repository** to Render.com.
2. Render will automatically detect the `render.yaml` blueprint.
3. Configure the following **Environment Variables**:

| Variable | Source |
|----------|--------|
| `DATABASE_URL` | Auto-generated by Render Postgres |
| `SECRET_KEY` | Auto-generated |
| `EMAIL_HOST_USER` | Your SMTP Email |
| `EMAIL_HOST_PASSWORD` | Your App Password |
| `TWILIO_ACCOUNT_SID` | Twilio Console |
| `TWILIO_AUTH_TOKEN` | Twilio Console |
| `RAZORPAY_KEY_ID` | Razorpay Dashboard |
| `RAZORPAY_KEY_SECRET` | Razorpay Dashboard |
| `OPENAI_API_KEY` | OpenAI Developer Portal |
