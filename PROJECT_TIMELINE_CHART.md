### Phase Descriptions:

1.  **Architecture:** Establishing the core Django structure, PostgreSQL database schemas, and JWT security.
2.  **Frontend/Catalog:** Designing the React UI and populating the database with thousands of products using automated discovery scripts.
3.  **Transactional:** Implementing secure shopping cart logic and full integration with the Razorpay API.
4.  **AI & Panels:** Integrating OpenAI GPT for the interactive bot and building comprehensive analytics dashboards for sellers.
5.  **QA:** Rigorous testing of the financial and notification meshes to ensure 99.9% reliability.
6.  **Launch:** Final deployment orchestration and production environment stabilization.
